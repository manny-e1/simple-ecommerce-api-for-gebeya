# simple-ecommerce-api-for-gebeya

## Technologies used

- Node with Express
- MongoDB

### To try it out

1.  clone the repo
2.  head to the root directory

#### with docker

run `docker-compose up`

#### without docker

run `npm install`
run `npm start`

### Deployed live link
